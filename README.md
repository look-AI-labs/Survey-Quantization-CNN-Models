# Survey-Quantization-DNN-Models
This is to quantize FP 8/4/2bits for CNN models. More codes on the way. 


### Dependency 
* Python 3.7
* PyTorch 1.1.0
* torchvision 0.2.1
* [tensorboardX](https://github.com/lanpa/tensorboardX)


### Train
#### Resnet-20 Models on CIFAR10
Just Run the script
 ```
 python train.py 
 ```
